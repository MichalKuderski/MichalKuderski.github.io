Featured Project

Healthcare Analytics — Medicare Provider Spending Analysis

Illinois · 2023 · CMS Open Data · SQL + Excel

Healthcare Analytics

End-to-end healthcare analytics project analyzing 2,460 CMS Medicare Physician & Practitioner claims records for Illinois psychiatry providers. Built a normalized PostgreSQL relational database, wrote six analytical SQL queries using CTEs, window functions, subqueries, and aggregations, and delivered an executive Excel dashboard with pivot tables and KPI summary sheet — simulating a full operational workflow for a community mental health organization.

A sixth original query examined telehealth access by city, surfacing that Elk Grove Village delivered 76.7% of its psychiatric services by telephone versus Chicago's 1.9% — pointing to transportation barriers and provider shortages in suburban communities that directly inform resource allocation and care delivery decisions.
